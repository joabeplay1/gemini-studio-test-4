# Gemini Studio - Gerador de Aplicativos

Aplicativo construido com Base44, powered by Gemini 2.5 Flash.

## Funcionalidades

- Gerador de prompts profissionais
- Pipeline de geracao completa
- Suporte a apps, sites e jogos
- Exportacao para ZIP
- Integracao com GitHub

## Estrutura

- `src/pages/` - Paginas
- `src/components/` - Componentes
- `base44/entities/` - Esquemas de dados
- `base44/functions/` - Funcoes backend

## Tecnologias

- React + Vite + Tailwind CSS
- Base44 SDK
- Gemini 2.5 Flash API
